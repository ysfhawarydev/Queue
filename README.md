# 🚶 Queue Implementation in Java (Array & LinkedList)

A **custom implementation of Queue data structure** in Java using both:
- Array-based approach
- LinkedList-based approach

This project is built using **Object-Oriented Programming (OOP)** principles to demonstrate how queues work internally without using Java built-in collections.

---

## 💡 What is a Queue?

A **Queue** is a linear data structure that follows the **FIFO (First In First Out)** principle:
> The first element inserted is the first one to be removed.

---

## ✨ Features

### 📦 Array-Based Queue
- ➕ Enqueue (insert element)
- ➖ Dequeue (remove element)
- 👀 Peek (view front element)
- 📊 Display elements
- ⚠️ Handle overflow and underflow cases

### 🔗 LinkedList-Based Queue
- ➕ Enqueue (insert at rear)
- ➖ Dequeue (remove from front)
- 👀 Peek front element
- 📊 Dynamic size handling
- ⚡ No fixed capacity limitation
